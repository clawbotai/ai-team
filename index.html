Discord Bot 集体失联

ERROR: 4014 Privileged Intent · Xiao Ma · Mar 2026

问题：5 个 Bot 全部不回复消息
原因：Message Content Intent 没开启
解决：Discord Developer Portal 开启 Message Content Intent

Bot 创建 checklist 必须包含 Intent 检查

专属频道配置无效

ERROR: BINDING_NOT_FOUND · Xiao Ma · Mar 2026

问题：专属频道发了消息，对应 Bot 没反应
原因：binding 配置只有 peer，没有 requireMention 设置
解决：添加 requireMention: false 到专属频道配置

binding 三要素：peer + requireMention + allow

统筹频道@人不回复

ERROR: MENTION_IGNORED · Xiao Ma · Mar 2026

问题：统筹频道@小码，小码不回复
原因：小码 Bot 没有绑定统筹频道
解决：给每个 Bot 添加统筹频道的 binding，requireMention: true

多频道配置要画矩阵图，避免遗漏

ignoreOtherMentions 踩坑

ERROR: CONFIG_NOT_FOUND · Xiao Ma · Mar 2026

问题：想找 ignoreOtherMentions 参数，文档里没有
原因：这个参数在 guilds 层级，不在 channels 层级
解决：在 guilds.YOUR_GUILD_ID 下添加 ignoreOtherMentions: true

配置层级要看官方文档的完整 schema

Token 明文泄露

ERROR: SECURITY_HIGH_RISK · Xiao Ma · Mar 2026

问题：安全审计发现 6 个 Discord Token 明文写在配置文件
原因：为了方便直接写死了
解决：创建.env 文件，所有 Token 移到环境变量

Token 永远不要明文提交到 Git

飞书配置完全开放

ERROR: DM_POLICY_OPEN · Xiao Ma · Mar 2026

问题：飞书配置 allowFrom: ["*"]，任何人都能访问
原因：测试时图方便
解决：改为 allowlist 模式，限制特定用户 ID

测试配置上线前必须 review 安全设置
