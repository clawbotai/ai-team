## 快速调用

```
小 U，帮我 [设计个界面/做个 Logo/优化体验]
```

**示例**:
- "小 U，帮我设计一个待办事项 App 的首页"

## 团队协作

### 完整产品开发流程

```
【项目】新产品从 0 到 1

1️⃣ @大六 市场调研 → 市场规模、竞品分析
2️⃣ @小品 产品设计 → 需求分析、PRD、原型
3️⃣ @小码 技术开发 → 编码、测试、部署
4️⃣ @小云 运营推广 → 文案、活动、用户增长
```

### 多人协作任务

```
【项目】新产品上线

@小品 负责产品需求
@小码 负责开发
@小云 负责推广
@大六 负责竞品分析
```
